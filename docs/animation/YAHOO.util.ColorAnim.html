ColorAnim subclass for color fading.

Usage: var myAnim = new Y.ColorAnim(el, { backgroundColor: { from: '#FF0000', to: '#FFFFFF' } }, 1, Y.Easing.easeOut);

Color values can be specified with 112233, #112233, #123, or rgb(255,255,255)
